Install and Configuration CI4

For install, only download and put in our root path

The configuration starting in:

  • .env file: This file contain the configurations of our app. Here we must changed our app url, database, and mode of app (production/development). Note important: This file must be duplicated for editing later. This file must have a name: .env ***important
  • app/config/App.php: Here we have the config initial for our app: as timezone, locale
  • app/config/Database.php: Here we can put our data for DB
  • app/config/routes.php: Here we can edit our routes for controllers and URI of our app
